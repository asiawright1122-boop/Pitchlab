## 0. 诚实提醒:世界杯是"最难赢"的市场,但"最适合做分发"

| 维度 | 现实 |
|---|---|
| 市场有效性 | 🔴 **全球下注量最大 → 盘口最锐**,最难有 CLV(与 §04"避开主流高效市场"相悖) |
| 样本量 | 🔴 国家队比赛少、阵容多变;仅 104 场,统计上薄 |
| 模型适配 | 🔴 俱乐部 Dixon-Coles **不适用**国家队(需国际赛 Elo 模型) |
| **流量/关注** | 🟢🟢 **史上最大体育关注**——这正是我们说的核心增长杠杆"分发" |
| 产品打磨 | 🟢 逼我们在真实激动人心的赛事上把"预测/报告/模拟/推送"全链路跑通 |

> **结论:把世界杯当作【分发 + dogfooding + 诚实校准展示】,不当作【edge 尝试】。**
> 我们**不宣称打穿世界杯盘口**;我们展示"透明、校准良好的概率 + 夺冠/出线模拟器 + 公开 CLV",用世界杯的巨大流量**建立信任与受众**(B 内容 + C 工具的冷启动)。edge 狩猎放到后续低级别联赛 niche。

---

## 1. 技术方案:国际赛模型(非俱乐部 Dixon-Coles)

```text
martj42/international_results (CC0, ~49k 场 1872–2026)
        │  拟合
        ▼
  国际赛 Elo 评分(含主场/中立场、赛事重要性、净胜球加权)
        │  Elo 差 → 预期进球
        ▼
  Poisson 比分矩阵 → 1X2 / 大小球 概率(复用 markets/derive.py)
        │  蒙特卡洛
        ▼
  锦标赛模拟器(12 组×4 → 每组前2 + 8个最佳第3 → 32强 → … → 决赛)
        │
        ▼
  输出:每队 出线概率 / 晋级各轮概率 / 夺冠概率  +  逐场比赛预测报告
```

**关键:** 复用已建的 `markets/derive.py`(比分矩阵派生市场)与 `metrics/`(CLV/Brier),只新增"国际赛 Elo + 锦标赛模拟器"。

## 2. 数据来源

| 用途 | 源 | 合规 |
|---|---|---|
| 国际赛历史(拟合 Elo) | martj42/international_results(CC0 开放) | 🟢 |
| 2026 赛程/分组 | football-data.org 免费(含 FIFA World Cup)或官方 | 🟢 |
| 赔率(可选,算 CLV) | The Odds API(soccer_fifa_world_cup)/ SportsGameOdds | 🟢 |

> **分组/赛程用 config 驱动,从官方源填充,绝不杜撰**(我提供结构 + 模板,真实分组由官方/football-data.org 灌入)。

## 3. 输出形态(分发友好)

- 🏆 **夺冠概率榜 / 出线概率**(最易传播,适合做内容)
- 📅 **逐场比赛预测报告**(概率 + 简析)
- 📊 **赛后校准追踪**(Brier / 命中)——展示诚实
- (有赔率时)**CLV 透明记录**——立信任
