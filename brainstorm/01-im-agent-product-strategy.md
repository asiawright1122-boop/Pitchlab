## 0. 一句话定位

> **一个面向海外英语球迷的 Telegram 原生 AI 看球搭子:发条消息订阅球队,持续收到赛程、实时盘口、AI 人话看点与预测;配「虚拟钱包模拟盘」按真实赔率竞猜、用 CLV 给眼光打分、和朋友开房比拼——不碰真钱、不导流下注,靠工具型 SaaS 订阅变现。**

与项目原定位的关系:这是 `D1′`(C 工具 + B 内容)+ `D13`(多渠道订阅推送)在**「大众化、IM 原生」外壳**下的具体化。底层量化引擎(DC/Elo/devig/CLV)不变,前台把术语翻译成人话。

## 1. 战略收敛路径

```
商业化&增长 → 出海·英语大众 → AI看球助手(说人话)
   → 股市式「模拟盘」(真实盘口 + 虚拟钱包竞猜,不碰真钱)
   → 区块链?→ 否决(只留密码学「可证明诚实」为远期可选)
   → 主交付形态 = IM 原生(Telegram 首发)
   → 留存设计 = 触发 × 投入 × 绑定
```

核心 reframe:**价值主张从「帮你赢钱」换成「帮你更懂球 + 乐子 + 社交」**。这化解了 Phase 0 的 No-Go(主流 1X2 无 +CLV edge),把「诚实/不忽悠」从劣势变成信任资产。

## 3. 四条合规红线(承重墙 — 动了全盘崩)

调研确认:整个合规姿态**完全依赖**虚拟币没有真实价值。必须同时满足:

1. 🔴 **虚拟币只送不卖 + 不可兑现 + 不可交易/转让**
   - UK《Gambling Act 2005》"money or money's worth" 测试 = 可兑现 **OR 可交易** → 不只禁提现,**还要禁用户间转让筹码**
   - Discord 四要件:虚拟币若不可兑换/交易,即不构成 "real-world value" → 不违规
2. **无现金 / 无实物奖**(排名只给徽章/面子/榜单)
3. **强制 18+ 年龄门**(澳洲 2024-09-22 起 simulated gambling 强制 R18+,**无真钱也不豁免**)
4. **不导流博彩、不做 affiliate、不抽水、措辞中性**(禁「稳赚/必中/包赢」)

> 收入只能来自**工具订阅**,绝不来自卖币——这反而把商业模型逼成更干净的 SaaS。

## 4. 调研核心结论(带引用)

deep-research:28 源 → 129 claim → 核查 25 条,**25 confirmed / 0 refuted**(多为一手政策/ToS,事实性强)。

### 4.1 IM 平台红线
- **WhatsApp 弃用**:Meta Business Policy §5 把 Real Money Gambling 列默认禁止的受监管垂类,定义极宽(任何货币价值的入场/奖品),仅约 5 国(澳/日/哥/墨/秘)经批准。赔率/预测产品在其重点审查区。〔business.whatsapp.com/policy〕
- **Discord 友好**:只禁**非法**赌博(四要件累积);虚拟币不可兑换/交易则不违规。〔discord.com/safety/gambling-policy-explainer〕
- **Telegram**:bot 能力最强、尺度最宽(本调研未逐字核查,沿用常识判断为首发最佳)。

### 4.2 数据源选型
| 源 | 价格 | ToS 对 IM 推送 | 结论 |
|---|---|---|---|
| **The Odds API** | 免费档 / 付费 ~$30–119 | ✅ **明确允许** user-facing app 商用,不可转卖原始 feed | **首选**〔the-odds-api.com/terms〕 |
| SportsGameOdds | 免费 Amateur(2.5k/mo)/ $99 / **$299 含 Pinnacle** | 含 Pinnacle 锐盘基准(Pro 档) | 备选;**修正项目 D4「免费含 Pinnacle」**〔sportsgameodds.com/pricing〕 |
| API-Football | — | ⚠️ 不授发布许可,IM 大规模推送或算「mass media distribution」需额外授权 | 摩擦高〔api-football.com/terms〕 |
| Sportmonks | 基础 €29/99/249 + 赔率预测 add-on ~€24 | 偏博彩运营商定位;ToS 是否允许待查 | 贵、定位偏〔sportmonks.com〕 |

### 4.3 竞品(活的近似品)
- **FootyLiveBot**〔footylive.org〕:Telegram Mini App,~20s 实时推送 + **无真钱积分竞猜** + **Telegram Stars freemium**,自报 ~17–21k 月活(未验证)。**验证了路线可行**;但订阅是**按钮式,非对话式** → 你的「对话订阅」差异化存疑(见 §8)。
- **Sports AI**〔sports-ai.dev〕:$6.99/mo + $34.99 终身 + 免费档,卖 "value betting";**ROI 宣称是营销非已证实 edge**(再次印证 Phase 0)。
- **OpenClaw**:Tom's Hardware 报道单月烧 $1.3M OpenAI token(未进正式核查但有来源)→ 坐实「自主 agent 每步过 LLM = 成本核爆」,支持自建精简编排器。

## 5. 盈亏测算摘要

**模型形状:高毛利、低固定成本、边际成本近零。**(内容单例共享 + Telegram 推送免费 + LLM 随比赛数不随用户数)

### 假设
定价 $6/mo · 转化 2%(基准)· 净 ARPU **$4.8**(扣 Stars/IAP 30% 与 Web 6% 加权)· churn 7%/mo

### 三场景(未计创始人工资)
| | 1万 MAU | 10万 MAU | 100万 MAU |
|---|---|---|---|
| 付费用户(2%) | 200 | 2,000 | 20,000 |
| 净收入/mo | $960 | $9,600 | $96,000 |
| 固定成本/mo | $550 | $1,250 | $5,000 |
| **净利/mo** | **+$410** | **+$8,350** | **+$91,000** |
| 年化 | ~$5k | ~$100k | **~$1.09M** |

### 命门
- **Break-even**:覆盖成本 ~5,750 MAU;养活全职创始人 ~4.7 万 MAU
- **每 MAU 终身价值 ≈ $1.3** → 🔴 **CAC 必须 < $1.3,买量死路 → 只能自然/内容/SEO 获客**
- **天花板** ~$1M/年(好的 small business,非暴利)→ 突破需二次变现(B2B 数据/API、白标、赞助)

> 结论:**成本不是问题,获客 × 转化是唯一命门。** MVP 第一个要证伪的是「能否低成本自然获客 + 转化撑到 2%」,而非「产品好不好用」。

## 6. 产品与留存设计 ⭐(本 brief 重点)

> **留存 = 触发(为什么回来)× 投入(为什么留下)× 绑定(为什么不走)**

### 触发:比赛节奏 = 内置回访器
每场球的「赛前→赛中→赛后」三节点,永不枯竭地制造回访理由:
| 节点 | 推送 | 用户为何来 |
|---|---|---|
| 赛前 T-2天 | 人话卡:看点/AI 预测/谈资 | 做功课 |
| 赛前 T-6h | 伤停 → AI 调整信心 | 看变数、改竞猜 |
| 赛中 | 进球/红牌 | 盯着看 |
| **赛后** | **AI 诚实复盘 + 你的竞猜 CLV 打分** | **对答案(最强钩子)** |

⚠️ 推送疲劳 = 卸载第一杀手 → 只推订阅内容 + 智能频率 + 可调。

### 投入:模拟盘 = 进度型留存(最强护城河)
- 未结算持仓(蔡格尼克效应)· 盈亏曲线(沉没成本)· 赛季+段位(周期目标)
- 🎯 **CLV 打分把「运气游戏」升级成「眼光成长游戏」**:输了但选得对 → "+4% CLV,眼光毒,只是运气差"。**用户关系从「赌徒-赌场」变「学员-教练」**,抗挫折、更长久。**只有你能做(有量化引擎),FootyLive 做不了。**

### 绑定:社交 = 留存放大器(利用 Telegram「群」)
- **群内 bot**:拉进已有球迷群,全群竞猜 → 病毒 + 留存二合一
- **好友私密房 / 联赛**:和朋友开房比拼 → 最强留存锁(不会单独退群)
- 1v1 对赌 · 好友榜 > 联赛榜 > 全球榜

### 情感层:AI 有「人味」
有人设的懂球搭子:记住你的球队、赛后调侃你的竞猜、像朋友聊球而非报表。**卸载工具容易,卸载「懂你的朋友」难。**

### 一个用户的一周(Hook 循环)
```
周二 推前瞻卡 → 周四 伤停变数改竞猜 → 周五 下注+好友对赌
→ 周六 赛中盯着 → 周日 AI复盘+CLV打分+好友榜 → 下周循环
```

### 留存→付费衔接
免费:订阅 3–5 队 + 基础模拟盘 + 1 个好友房(养成习惯)。付费 $6/mo:无限球队 + 深度 AI 解说 + CLV 高级分析 + 多账户 + 私密房扩容 + 历史回看。**付费墙卡在「已投入、舍不得」处,不卡入口。**

### 三个留存杀手
1. 推送疲劳 → 频率控制 2. 空窗期(国际比赛日/夏歇)→ 非比赛日内容(转会/历史/AI问答) 3. 新手太难 → 前两周用信心⭐代替赔率,渐进揭示

## 6.5 Agent 设计与架构分层

**定位**:Agent 不用来「点菜」(简单操作卡片/按钮更快),而用来「陪你懂球」——开放问答 + 记忆 + 个性化 + 人味。**这才是甩开 FootyLive(按钮+积分)的真差异化。**

| 用户想干的事 | 最佳交互 |
|---|---|
| 订阅球队 / 竞猜 / 看结果 | **卡片+按钮**(高频简单,点 > 说 > 打字) |
| 开放查询("有什么冷门""曼联怎么了") | **对话 / 语音** |
| 记住你是谁、爱押什么 | **记忆** |

**逐项判断**:
- **IM 即全部界面** ✅ = bot 消息 + 卡片+按钮 + **Telegram Mini App**(模拟盘/排行榜等复杂界面);**不用单独做 Web/App**
- **下注 = 卡片** ✅ 人话卡内嵌一键竞猜(FootyLive 已验证)
- **记忆** ✅ 产品级(画像/偏好/竞猜史/CLV 轨迹,确定性存 DB,核心)+ 技能级(解说 skill 沉淀,`D14`)
- **语音 = STT 输入平权入口** ✅ 凡能打字做的都能语音做;成本低(STT-only ~$0.001/条,可自托管近零);⚠️ 球队名×多口音易错 → **名称映射库纠错 + 不确定时回确认卡兜底**
- **自我学习进化(必须拆两半)**:✅ **体验进化**(越来越懂你/好用 = 画像+skill 复利,护城河)· 🔴 **预测进化**(Phase 0 已证伪主流联赛无 +CLV;LLM 自由进化破坏可回测,违 `D8`)→ **"越来越懂你"可以,"越来越神准帮你赢"不行**

**架构分层(守 `D14`:Agent = 外壳,内核确定性)**:
```
┌─ 交互层(Telegram)── 消息 · 卡片+按钮 · Mini App · 语音(STT 输入)
├─ Agent 编排层 ────── 意图解析 · 开放问答/解说 · 调记忆与工具
│                      ⚠️ 简单操作走规则/按钮零 LLM,仅开放查询过 LLM
├─ 记忆层 ──────────── 产品级(画像/偏好/竞猜史/CLV)+ 技能级(skill 沉淀)
├─ 确定性内核(不进 LLM)── DC/Elo 概率 · devig · CLV 打分 · 模拟盘结算
└─ 进化 ────────────── ✅ 确定性再训练/校准 + 画像/skill 复利
                       ❌ 不做 LLM 自由进化预测(Phase0 证伪 + 不可审计)
```

## 7. MVP:最小留存闭环

> **订阅球队 → 赛前人话卡 → 一键虚拟币竞猜 → 赛后诚实复盘 + CLV 打分 → 周榜**

跑通这条 Hook 循环即可验证留存。社交(群 bot/私密房)、AI 深度人设为第二步放大器。
**北极星指标:周活跃竞猜用户数(WAP)**——一指标串起「看卡→竞猜→对答案」全循环。

最该先押注:**CLV 打分(独有差异化)+ 好友私密房(留存放大器)**。
